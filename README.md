# FOFADON App

鳳妃堂 Android APP

## How to Install

```bash
yarn
```

## UI Kit

請見 `http://116.228.6.58:9082/fofadon/fofadon-onsan-uikit`

## How to Build

```bash
yarn build
cordova platform add android
#or cordova prepare
yarn apk
# or yarn debug
```
